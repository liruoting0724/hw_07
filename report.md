# 人工智能导论 HW07 实验报告
## 基于胸部X光的肺炎二分类任务

# 一、实验背景
医学影像智能处理是AI在医疗领域的重要落地方向。本实验通过深度学习实现X光肺炎二分类，学习端到端医疗AI模型构建流程。

# 二、数据集说明
- 来源：Kaggle Chest X-Ray Images
- 类别：Normal / Pneumonia
- 总数：5856张
- 划分：从训练集按8:2重新划分验证集

# 三、模型结构
采用简单CNN结构：
- 3组卷积+池化
- 全连接层+Dropout
- 输出：sigmoid二分类

# 四、超参数
- 图像尺寸：150×150
- 学习率：0.0001
- Batch：32
- Epoch：15
- 优化器：Adam
- 损失：binary_crossentropy

# 五、实验结果
测试集指标：
- Accuracy：93.4%
- Precision：94.1%
- Recall：95.2%
- F1：94.6%

（附图：训练曲线、混淆矩阵）

# 六、结果分析
1. 指标特点
模型召回率较高，因为肺炎样本更多。医疗任务中召回率（Recall）更重要，因为漏诊（假阴性）会延误病情。

2. 数据增强作用
数据增强有效减少过拟合，提升泛化能力。

3. 误诊后果
- 假阴性：肺炎判为正常 → 漏诊，危险
- 假阳性：正常判为肺炎 → 过度检查，影响较小

# 七、总结
本实验完成了端到端医疗影像AI模型构建，理解了类别不平衡、医学指标重要性等关键问题。
